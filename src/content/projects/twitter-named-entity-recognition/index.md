This was one of project for the HSE NLP course on Coursera. I was tasked to build a named entity recognition model that will extract the various entities present in the tweet. I used Bi-Directional LSTM's and have also tested the same by using GRU's in place of LSTM's. It was a fun project where I learned the underlying mechanisms of how to recognize things in a text.
