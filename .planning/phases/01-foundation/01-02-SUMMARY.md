## Decisions Made

- **`astro add react` + `astro add tailwind` over manual install**: The Astro CLI handles peer-version pinning, config-file mutation, and tsconfig JSX flag injection atomically. Cheaper and more correct than `npm install + hand-edit`.
- **Hand-edit `astro.config.mjs` after the CLI ran**: The CLI emits a minimal `defineConfig({ integrations: [react()], vite: { plugins: [tailwindcss()] } })`. Pattern 1 adds `site`, `output: 'static'`, and the source-attribution comment header on top. Hand-edit lets us hit Pattern 1 verbatim without fighting the CLI.
- **Installed `@astrojs/check@^0.9` + `typescript@^5.9` as devDependencies in this plan**: Plan 01 did not install them (Plan 01 Task 4 chose `strict` not `strictest`, did not run `astro check`). They are listed in this plan Task 1 Step D and are required for the FOUND-02 gate (`astro check` exit 0). Classified as Rule 3 (Blocking) auto-fix.
- **Deleted Plan 01 recovery leftovers (`astro-README.md.tmp`, `astro.gitignore.tmp`, `package.json.astro`) rather than adding them to tsconfig exclude**: The files were stale temp artifacts from Plan 01's D-04 recovery flow; they had no live consumer. Cleaning them out is more honest than papering over with exclude rules.
- **Did NOT install Vitest, ESLint, Prettier, husky, lint-staged in this plan**: Per plan Task 1 Step D explicit exclusion ("Plans 04 and 05 own those").
- **Did NOT run `npm audit fix --force` on the 5 moderate vulnerabilities reported after `npm install`**: Force-fix risks breaking-change upgrades. Vuln triage belongs in Plan 04 or a deferred-items entry, not this configuration plan.
- **Did NOT re-add `compilerOptions: { jsx: "react-jsx", jsxImportSource: "react" }` to tsconfig after overwriting to Pattern 8**: Pattern 8 is the documented target for this plan and does not include JSX overrides; Plan 03's hydration fixture is the first .tsx file and `astro check` will tell us if the strictest default needs them. Astro 6 + React 19 picks up JSX through `@astrojs/react`'s own type emit in most paths.
